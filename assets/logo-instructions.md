# Logo Setup Instructions

The logo images have been provided. To use them in the app:

1. Save the black/white logo (second image) as `logo.png` in this directory
2. The logo will be used in the sidebar and login page
3. Recommended size: 200x200px (circular crop with white background)

For best results:
- Use a circular crop of the logo
- Ensure white/transparent background
- Save as PNG format for transparency support
